Detailed Description

Controller principale.

La "mente" del progetto, attraverso il controller vengono effettuate tutte le operazioni logiche, sui dati messi a disposizione dalle altre classi. Include gestione "Lazy Loading": se ci sono troppi dati, le tabelle di visualizzazione partono vuote.

Costruttore del controller

Inizializza lo stage che viene passato come parametro. Si occupa di effettuare il caricamento del file dati all'avvio del programma (caricaTutto) e si occupa di effettuare il salvataggio automatico in caso di tentativo di chiusura del programma (salvaTutto)
